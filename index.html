How to Play

  • Sort the books so each shelf contains only one color.
  • Tap a shelf to pick up the rightmost book(s).
  • Books of the same color move together as a volume.
  • You can only place books on an empty shelf or next to a book of the same color.
  • Each shelf can hold a maximum of 7 books.
